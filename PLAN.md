Photographer Portfolio Website – Project Plan

Goal

Build a minimalist, high-end portfolio website for a photographer specializing in Digital and Film photography. The site should feel premium, fast, and distraction-free, acting as a digital gallery that drives visitors to Instagram for contact.

⸻

Core Principles
	•	Extremely simple navigation
	•	Full focus on imagery
	•	Fast load times
	•	Mobile-first
	•	Editorial / gallery aesthetic
	•	No clutter

⸻

Navbar

Only three links:
	•	Film → /film
	•	Digital → /digital
	•	Contact → external link to Instagram profile

No logo clutter — name centered or left aligned.

⸻

MVP (Must-Have)

Pages

Page	Purpose
/	Redirect to /film or /digital (default gallery)
/film	Film photography gallery
/digital	Digital photography gallery


⸻

Gallery Layout
	•	Full-width responsive grid
	•	Masonry or justified layout
	•	Lazy loaded images
	•	Click → full screen lightbox view
	•	Optional subtle caption on hover

⸻

Lightbox
	•	Full viewport
	•	Keyboard navigation
	•	Swipe on mobile
	•	Minimal chrome

⸻

Contact
	•	Navbar Contact → external Instagram profile
	•	No internal contact form for MVP

⸻

Stretch / Phase 2
	•	Image categories / tags
	•	About modal
	•	Dark / light toggle
	•	Featured collections
	•	Print store integration

⸻

Tech Stack

Framework	Next.js (App Router)
Styling	TailwindCSS
Animations	Framer Motion
Images	Next Image / Cloudinary
Hosting	Vercel
CMS (optional)	Notion / Sanity (future)

⸻

Design Direction
	•	White or off-white background
	•	Black text
	•	Large spacing
	•	Serif or editorial font (Playfair / Cormorant / Canela)
	•	Smooth fade-in animations
	•	Very light hover transitions

⸻

Cursor Implementation Notes
	•	Prioritize Lighthouse score 95+
	•	Images must lazy load
	•	Mobile performance first
	•	No unnecessary dependencies

⸻

Success Criteria
	•	Loads under 2 seconds
	•	Visitors can view work in <2 clicks
	•	Instagram link is visible at all times
	•	Site feels like a premium art gallery

⸻

North Star

“This should feel like walking into a modern photography gallery — silent, focused, and intentional.”
